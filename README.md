# Streamerbot_giveaway_script
A streamer.bot script that allows the streamer to start and end giveaways with chat commands

# Installation
- Download the "giveawayfunction_import_file" from the releases section or use this link -> https://github.com/itshotlatte/Streamerbot_giveaway_script/releases/tag/streamer.bot
- Open streamer.bot
- Click on "import" on the top bar
- Select the "giveawayfunction_import_file", all the actions should import in to streamer.bot
- Make changes to the "excute code" subactions to change the chat messages sent by streamer.bot
